“Game Milestones” is an interactive installation in Berlin’s Computer Game Museum. On a wall of cubes, each cube represents a computer game which is outstanding in the development of this young genre.
The visitor is able to steer a green cursor from cube to cube to reveal text, image and video information on the games.
The crosshair design and the intensity of the projection (via a moveable projector) make this archetypical UI Element traverse into the reality of the museum space.

Eine Wand aus Kuben („Pixel“) repäsentiert die innovativsten und einflussreichsten Computerspiele, die Meilensteine, in chronologischer Reihenfolge.
Museumsbesucher können tiefergehende Informationen und Videomitschitte des Spielverlaufs anzeigen lassen, indem sie einen Lichtcursor mittels eines Joysticks über die Wand steuern. Die Informationen werden dann über zwei in die Ausstellungsarchitektur integrierte Monitore angezeigt. Auch das Licht in zehn eingebetteten Vitrinen wird vom Cursor aktiviert.
Der Leuchtende Cursor, ein archetypisches Auswahl-Werkzeug, stellt durch seine Manifestation in der physischen Welt einen Kommentar zum Trend der „Augmented Reality“ dar, eine Technik, die mehr und mehr auch in Spielen für Mobile Geräte Anwendung findet. Auf diese Weise kündigt die Installation bereits den nächsten Meilenstein an.
