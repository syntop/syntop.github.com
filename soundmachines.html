Three units, which are resembling standard record players, translate concentric visual patterns into control signals for further processing in any music software.

The Soundmachines premiered on the Volkswagen New Beetle stand at the IAA motor show in late Summer 2011. In cooperation with the sounddesigner/ producer Yannick Labbé of TRICKSKI fame, we developed three unique discs, each controlling one track of an Ableton Live Set exclusively made for the Event. The show was supported by a set of realtime generated visuals, running on a 25m wide LED wall.

Drei Einheiten, normalen Plattenspielern ähnlich, übersetzen konzentrische Muster in Kontrollsigale.

Die Soundmachines kamen im Spätsommer 2011 auf dem VW-Messestand zum Einsatz. In Zusammenarbeit mit dem Sounddesigner und Produzenten Yannick Labbé (Trickski) entwickelten wir drei verschiedene Platten, von denen jede eine Spur eines exklusiv für diesen Event produzierten Ableton Live Sets ansteuert.
Die Performance wurde visuell von generativen Visuals auf einer 25m langen LED-Wand unterstützt.
